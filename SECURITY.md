# Security Policy

## Supported Versions

当前仓库默认只维护 `main` 分支上的最新版本。

## Reporting a Vulnerability

如果你发现了安全问题，请不要直接公开提交包含利用细节的 Issue。

建议提供以下信息：

- 影响范围
- 复现步骤
- 可能泄露的数据类型
- 相关日志或截图

在问题公开修复前，请避免把敏感样本、账号令牌或完整系统日志直接贴到公开讨论中。

## Project Security Notes

- 本项目不设计远端账号体系，也不上传设备遥测数据。
- 历史数据默认只保存在本机用户目录。
- 需要管理员权限的数据采样为显式触发，不做默认后台持续提权。
